---
title: "updateMetadata"
---

Update the metadata for an existing file

## Usage

```typescript
import { PinataSDK } from "pinata";

const pinata = new PinataSDK({
  pinataJwt: process.env.PINATA_JWT!,
  pinataGateway: "example-gateway.mypinata.cloud",
});

const update = await pinata.updateMedatadata({
  cid: "bafkreih5aznjvttude6c3wbvqeebb6rlx5wkbzyppv7garjiubll2ceym4",
  name: "Pinnie V2",
  keyValues: {
    whimsey: 200
  }
})
```

## Returns

```typescript
OK
```

## Parameters

### cid

- Type: `string`

CID of the file you want to update

```typescript
const update = await pinata.updateMedatadata({
  cid: "bafkreih5aznjvttude6c3wbvqeebb6rlx5wkbzyppv7garjiubll2ceym4",
  name: "Pinnie V2",
})
```

### name (Optional)

- Type: `string`

Update the name of a file that will appear in `listFiles`

```typescript
const update = await pinata.updateMedatadata({
  cid: "bafkreih5aznjvttude6c3wbvqeebb6rlx5wkbzyppv7garjiubll2ceym4",
  name: "Pinnie V2",
})
```

### keyValues (Optional)

- Type: `object`

Update the `keyValues` object for a file

```typescript
const update = await pinata.updateMedatadata({
  cid: "bafkreih5aznjvttude6c3wbvqeebb6rlx5wkbzyppv7garjiubll2ceym4",
  keyValues: {
    whimsey: 200
  }
})
```
